# Boocap — Product Context

## 한 줄 정의
월별 독서 기록을 멋진 포스터로 시각화해서 SNS에 공유할 수 있게 해주는 웹 툴.

## 이름
**Book + Recap = Boocap**

---

## 배경 & 동기

책을 자주 읽는 사람들은 매월 자신이 읽은 책을 돌아보는 행위를 자연스럽게 한다.
지금까지의 방식은 독서 기록 앱 화면 캡처하거나 책들을 모아 사진 찍는 정도.

이 행위 자체가 "나의 데이터"인데, 더 다양하고 멋진 형태로 표현할 수 있어야 한다는 문제 의식에서 출발.
보통 사람들은 데이터를 활용해 시각적 결과물을 만드는 걸 어려워하므로, 그 과정을 Boocap이 해결해준다.

---

## 타겟 사용자

- 독서를 꾸준히 하는 사람
- 읽은 책 기록을 SNS에 공유하고 싶은 사람

---

## 현재 상태 (2026년 4월 기준)

- 개인 사이드 프로젝트
- 웹 기반, Vercel에 배포 중 (운영 중)
- 앱 전환 계획 없음 — 웹 툴로 유지할 가능성 높음
- 앞으로 홍보를 본격적으로 시작하고 사용자 피드백을 받아볼 예정

---

## 핵심 플로우 (3단계)

1. **Mark Your Days** — 달력에 날짜별로 읽은 책 등록 (최대 12권/월)
2. **Frame Your Month** — 템플릿 & 무드 선택
3. **Archive Visualized** — 포스터 이미지 다운로드

---

## 템플릿 (8종)

| ID | 이름 | 특징 |
|---|---|---|
| `grid` | Monthly Calendar | 균일한 그리드 |
| `grid2` | Borderless Calendar | 무테 달력, 종이 질감 |
| `stack` | Big Cover | 뉴스레터 스타일 |
| `list` | The Receipt | 리스트 타이포그래피 (≤9권) |
| `essay` | The Typo Slash | 영문 산문형 (≤5권) |
| `capsule` | Book Edge | 캡슐 안에 번호+제목 (4–8권) |
| `mosaic` | Floating Cover | 표지 중심 컬러 그리드 |
| `timeline` | Timeline | 세로 타임라인 (≥3권) |

## 무드 (4종)

| ID | 특징 |
|---|---|
| `minimal` | 밝고 깔끔한 기본 톤 |
| `editorial` | 크림 바탕, 골드 액센트 |
| `bold` | 다크 바탕, 오렌지 액센트 |
| `dark` | 거의 블랙 바탕, 무채색 |

## 기술 스택

- **Frontend**: React 18 + TypeScript + Vite
- **UI**: Tailwind CSS + shadcn/ui (Radix UI)
- **라우팅**: React Router v6
- **배포**: Vercel (+ Vercel Analytics)
- **언어**: 한국어 / 영어 (i18n 자체 구현)
- **이미지 생성**: html-to-image, dom-to-image-more

---

## 향후 고민 중인 방향

- 템플릿 스타일 다양화
- 포스터 개인 커스터마이징 기능 (색상, 텍스트 편집 등)
- 1년치 독서 데이터 시각화 (연간 리캡)
- 외부 독서 기록 서비스에서 데이터 불러오기 (예: 밀리의서재, 리디, 굿리즈 등)
